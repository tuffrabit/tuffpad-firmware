# tuffpad-firmware
Firmware for the TuFFpad.
 ## Flashing Instructions
 1. Download a release that contains a firmware hex file, teensy.exe, and teensy_reboot.exe.
 2. Plug your tuffpad into your PC.
 3. Run teensy.exe.
 4. Click "Open Hex File" from the File menu.
 5. Choose the downloaded hex (tuffpad.hex for retail or tuffpad_alpha.hex for a test-alpha unit) file.
 6. Run teensy_reboot.exe.
 7. You should see the teensy.exe application inform you it's flashing the device.
 8. Your tuffpad should auto reboot and will be running the updated firmware.
